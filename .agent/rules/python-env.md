These instructions are only for Python side if the project.

- The project should be structured with poetry, pyproject.toml, and uv.
- Any dependency install, should use poetry or uv to install things. 
- To create the virtual env if it does not exist, use uv.
- try to include the latest version of any package that you use, clearly mention the version in pyproject.toml. 
- the project should as a package on python side.
- Use fastapi and software engineering best practices.
- When writing code, don't use emojis and special characters for logging
